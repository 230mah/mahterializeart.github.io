# Mahterialize Art Store

Loja online para venda de estátuas artesanais e itens colecionáveis.
Projeto desenvolvido em HTML, CSS e JavaScript e hospedado via GitHub Pages.

Diretório:
- index.html – página inicial da loja
- assets/css/style.css – estilos do site
- assets/js/script.js – scripts do site
- produtos/ – imagens e páginas individuais dos produtos
